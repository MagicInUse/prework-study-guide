HTML

  • The head element contains information about the webpage.
  • The meta element is used for multiple items (such as charset, content & viewport) to help with SEO (Search Engine Optimization).
  • The body element represents the visible content shown to the user.
  • Other HTML tags (such as header, main, section & footer) are used to structure specific items on the webpage.
  • The h1 tag should only be used once per page, and then numerically in order of importance (h2, h3, etc).

CSS

  • "*" is used to apply a style to all elements.
  • "." is used to apply to a class.
  • "#" is used to apply to an id.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status : checks what branch we are currently on.
  • touch file-name : creates a file, extension must be included.
  • git checkout -b branch-name : creates a new branch and switches to it.
  • git add -A : Used when we want to commit all files in the working branch, before the commit command.
  • git commit -m "comment" : commits to the repository, -m "comment" adds a comment to the commit.
